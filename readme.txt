1. Download the Repository

    Go to the repository's master branch.
    Download the project as a ZIP file.
    Extract the ZIP file into a folder of your choice.

2. Navigate to the Folder

    Open a terminal and navigate to the extracted project folder.

4. Install Dependencies

    Run the following command to install all required dependencies:
    ' npm install '

    If any dependencies are missing, install them manually using this command:
    ' npm install @emotion/react@11.13.5 @emotion/styled@11.13.5 @mui/icons-material@6.1.9 @mui/material@6.1.9 @reduxjs/toolkit@2.3.0 formik@2.4.6 react@18.3.1 react-dom@18.3.1 react-redux@9.1.2 react-router-dom@7.0.1 yup@1.4.0 '

5. Start the Development Server

    Launch the application using
    ' npm run dev '

    This command will start the development server, and you can view the website in your browser.
